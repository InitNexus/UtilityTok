### Introduction
Welcome to the UtilityTok repository. UtilityTok is a platform orientated around TikTok and includes various OSINT, analytics and statistics tools as well as a public database. The main goal of the platform is to provide users with tools that serve several purposes such as:

- Pointing out key factors on why an account is doing better or worse than others
- Helping agencies learn more about their members
- Unmasking trolls and preventing abuse and bullying
- Providing additional stools and features to the creators who take TikTok seriously

> [!NOTE]
> Please note that this repository only contains the front-end code for UtilityTok. There is no back-end functionality and this repository serves as a backup. You can find the full website [here](https://google.com/404).

---

### Tools
UtilityTok includes various tools that serve different purposes:

ANALYSIS
- Basic Analysis ```Gather profile, account and general statistics information```
- Advanced Analysis ```Gather profile, account, authorization, domain, general statistics, advanced analytics and miscellaneous information```

INFORMATION LOOKUP
- Authorization Information ```View if an account has the following tied to it: email address, phone number, password, passkey & OAuth / third party platform```
- Domain Information ```View the email address domain and phone number area code tied to an account```

VIEWER
- Posts Viewer ```View posts along with relative information as well as download them```
- Liked Posts Viewer ```View liked posts along with relative information as well as download them```
- Favourite Posts Viewer ```View favourite posts along with relative information as well as download them```
- Re-Posts Viewer ```View re-posts along with relative information as well as download them```
- Stories Viewer ```View stories along with relative information as well as download them```
- Highlights Viewer ```View highlights along with relative information as well as download them```
- Stickers Viewer ```View stickers along with relative information as well as download them```
- Community Gifts Viewer ```View community gifts along with relative information as well as download them```
- TikTok Gifts Viewer ```View TikTok's live stream gift animations along with relative information as well as download them```
- Live Stream Viewer ```View live streams anonymously as well as all comments and any gifts that get sent```
- Profile Viewer ```View profiles anonymously without the other person knowing```

TRACKER
- Link Tracker ```Check who created a link, who shared it and who interacted with it. You can also see the operating system of each individual```
- Activity Tracker ```Check if a user is currently online, how long they've been online for or when they were last online```
- Profile Updates Tracker ```Track updates made across profiles```

CREATOR
- Post Creator ```Create posts with additional tools that TikTok doesn't provide```
- Story Creator ```Create stories with additional tools that TikTok doesn't provide```
- Highlight Creator ```Create highlights with additional tools that TikTok doesn't provide```
- Stickers Creator ```Create stickers with additional tools that TikTok doesn't provide```
- Community Gift Creator ```Create community gifts with additional tools that TikTok doesn't provide```

MISCELLANEOUS
- Link Shortener ```Shorten any TikTok URL without metadata```
- Alternate Account Detector ```Check if the person behind one account has more logged in on the same device```
- Media Downloader ```Download any type of TikTok media in any quality and without watermarks```

### Database
UtilityTok has a public database of collected information on random profiles. This includes the following information:  

PROFILE INFORMATION
- Visibility
- Nickname
- Nickname Edited
- Profile Picture
- Profile Picture URL
- Biography
- Biography / Profile URL's

ACCOUNT INFORMATION
- Creation Timestamp
- Username
- Username Edited
- User ID
- SecUID
- User URL
- Created Region
- Locked Region
- Current Region

LOGIN INFORMATION
- Email Address
- Phone Number
- Passkey
- Password
- Authorization
- Authorization Platform

DOMAIN INFORMATION
- Email Domain
- Number Domain

GENERAL STATISTICS
- Followers
- Following
- Friends
- Posts
- Total Likes
- Interactions
- Gifter Level

MISCELLANEOUS INFORMATION
- Stitch
- Duet
- Comments
- Live Room ID
- Follower List Visibility
- Following List Visibility

ADVANCED STATISTICS
- Includes analytics that cover average and individual statistics across all posts and media

> [!NOTE]
> Please be aware that no personal or private information as well as credentials are included in any of the database listings.
